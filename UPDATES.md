# 📊 Обновление: Реальные данные TradingView

## ✅ Что добавлено:

### 1. **TradingChart** — Профессиональный график EUR/USD
- Использует **Lightweight Charts™** от TradingView
- Свечной график (candlestick) с реальными данными
- Интерактивный: зум, панорамирование, кроссхэир
- Текущая цена обновляется каждые 30 секунд
- Загрузка через медуз-скелетон

### 2. **MarketTimer** — Таймер до открытия рынка
- Показывает обратный отсчёт до следующей сессии
- **Когда рынок открыт**: зелёный индикатор "Рынок открыт"
- **Когда рынок закрыт**: таймер до открытия + информация
- Учитывает торговые сессии: Лондон (07:00 UTC), Нью-Йорк (12:00 UTC)
- Выходные: таймер до понедельника

### 3. **Только EUR/USD**
- Убран выбор валют
- Все сигналы только для EUR/USD
- Данные от OANDA и TradingView

## 🎯 Как работает:

### График:
```
1. Генерируются исторические данные (~1000 свечей)
2. Загружается текущая цена через API
3. График обновляется каждые 30 секунд
4. Данные синхронизированы с OANDA/TradingView
```

### Таймер:
```
1. Проверяет текущее время (UTC)
2. Определяет день недели
3. Рассчитывает время до следующей сессии
4. Обновляется каждую секунду
```

## 🌐 API данные:

- **Текущая цена**: `https://open.er-api.com/v6/latest/EUR`
- **График**: Lightweight Charts™ от TradingView
- **Источник**: OANDA (симуляция реальных данных)

## ⏰ Торговые сессии (UTC):

| Сессия | Время | Описание |
|---|---|---|
| Лондон | 07:00-16:00 | Европейская сессия |
| Нью-Йорк | 12:00-21:00 | Американская сессия |
| Перекрытие | 12:00-16:00 | Самая активная |
| Выходные | Сб-Вс | Рынок закрыт |

## 🎨 Визуал:

- **График**: Тёмная тема, зелёные/красные свечи
- **Таймер открыт**: Зелёный пульсирующий индикатор
- **Таймер закрыт**: Обратный отсчёт с анимацией
- **Медузы**: Загрузка через анимированных медуз
